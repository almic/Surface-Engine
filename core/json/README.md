# surface-json

Standalone JSON parsing and manipulation library.

Follows the rfc8259 specification here: https://datatracker.ietf.org/doc/html/rfc8259.
<br>
But generally, the implementation of the parser is just the original https://www.json.org spec.

This library is part of the Surface Engine Core Libraries: https://github.com/almic/Surface-Engine
